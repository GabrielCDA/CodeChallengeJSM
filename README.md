# CodeChallengeJSM
Code Challenge Justos Somos Mais

# O Projeto
O projeto foi feito utilizando .NET Core. Foi implementado o swagger para ter uma interface mais amigável para a utilização.
Foi dividido em modulos, para caso seja reaproveitado as bibliotecas em outro projeto. Por exemplo, utilizar a biblioteca de estrura em algum outro projeto.
Foi feito apenas um método Get para devolver os clientes de acordo com os parametros passados.

# Executando o projeto
Basta executar a solution e passar os parametros (todos opcionais) através do swagger. O retorno será dado pelo proprio swagger, em uma interface amigavel.
